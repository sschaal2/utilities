Note
draw_data_point
Date
January, 1994
Remarks

draws a new training data point according to various critera

Function Parameters: [in]=input,[out]=output

Parameters
[in]X: the data matrix with input data
[in]Y: the data matrix with output data
[in]n_data: number of data points in matrices
[in]n_in: number of inputs
[in]n_out: number of outputs
[in]index_function: index function to be used for GENERATING_FUNCTION
[in]sampling_method: RANDOM_SAMPLING, SEQUENTIAL_SAMPLING, GENERATING_FUNCTION, PSEUDO_RANDOM_SAMPLING
[in]ind: for pseudo random sampling ptr to an index array. Pass a variable initialized with NULL to make this program assign it appropriately
[out]x: the output vector
[out]y: the input vector
[out]current_data_point: the current data point
